# 컬렉션
- 자료구조를 구현한 클래스
- 리스트, 스택, 큐, 집합, 해시테이블 등
- 컬렉션 인터페이스와 컬렉션 클래스로 나누어 제공
- 모두 제네릭 기능 지원

## Vector
- 가장 오래된 자료구조 클래스
- 가변 크기의 배열을 구현
- 요소 개수 늘어나면 자동으로 배열 크기 늘어남
- 어떤 타입의 객체도 저장


### 인터페이스
- Collection: 모든 자료구조의 부모 인터페이스, 객체의 ㅁ임
- Set: 집합을 나타내는 자료구조
  - HashSet: 키값만 중복 안 되게
  - LinkedHashSet: 넣는 순서대로
  - TreeSet: 정렬
- List: 순서가 있는 자료구조로 중복된 원소 가능
  - ArrayList: 순차추가삭제는 빠르지만 중간추가삭제는 느림
  - LinkedList: 개수가 계속해서 변하면 LinkedList를 사용
- Map: 키와 값으로 구성(사전형태)
    - HashMap: 키값만 중복 안 되게
    - LinkedHashMap: 넣는 순서대로
    - TreeMap: 정렬
- Queue: 들어오는 순서대로 나가는 자료구조
